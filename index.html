Bookmarklet Forge

Bookmarklet Forge is a tiny browser tool that opens a sidebar and helps you create reusable bookmarklets for the current webpage.

Install

Drag this button to your bookmarks bar:

Bookmarklet Forge

How to use

  1. Drag the blue Bookmarklet Forge button to your bookmarks bar.
  2. Go to any webpage.
  3. Click the bookmarklet.
  4. The Bookmarklet Forge sidebar should open.
  5. Choose a template or describe what you want.

Manual bookmarklet URL

If dragging does not work, create a new bookmark manually and use this as the URL:
